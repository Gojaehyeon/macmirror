모든 프리셋은 사파리 안전영역(Dynamic Island·홈 인디케이터·툴바) 제외한 실제 표시 가능 영역 기준. 메뉴바에서 방향(세로/가로)·HiDPI(Retina 렌더링) 토글로 즉시 변경.

이렇게 동작합니다

맥에 비공개 CGVirtualDisplay API 로 가상 디스플레이를 만들고, ScreenCaptureKit 으로 30fps JPEG 캡처. 아이폰은 HTTP 폴링으로 받아서 풀스크린 렌더.

설치

맥쪽 메뉴바 앱 + 아이폰쪽 네이티브 앱.

1

레포 클론

# 어디든
git clone https://github.com/Gojaehyeon/macmirror.git
cd macmirror
2

맥 앱 빌드

Xcode 16 이상. 처음 실행 시 화면 기록 권한 요청.

cd mac
./build-app.sh
open macmirror.app
3

iOS 앱 빌드

xcodegen + Xcode. 본인 Apple ID로 서명해서 폰에 배포.

cd ../ios
xcodegen generate
open MacMirror.xcodeproj
# Xcode에서 ⌘R
4

주소 입력

iOS 앱 가운데 탭 → 맥 메뉴바의 LAN 주소 입력 → 연결.

# 예시
192.168.0.10:8890

한계 / 알아두실 점

  • 비공개 API: CGVirtualDisplay 는 비공개. macOS 메이저 업데이트로 깨질 수 있음.
  • System Settings UI: 너무 작은 가상 디스플레이는 macOS 26.5 의 디스플레이 설정 UI 에서 숨겨질 수 있음. 프리셋은 모두 노출되는 크기로.
  • 무료 Apple ID: 7일마다 재서명 필요.
  • 요구 사항: macOS 14+, iOS 17+, Apple Silicon 권장, Xcode 16+.
